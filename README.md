# KotlinAndroidUtils
KotlinAndroidUtils


### Dependency

Add this to your module's `build.gradle` file (make sure the version matches the last [release](https://github.com/humazed/RoomAssety/releases/latest)):

Add it in your root build.gradle at the end of repositories:

```gradle
allprojects {
	repositories {
		...
		maven { url "https://jitpack.io" }
	}
}
```

Add the dependency
```gradle
dependencies {
    // ... other dependencies
    implementation 'com.github.humazed:KotlinAndroidUtils:{latest-version}'
}
```
